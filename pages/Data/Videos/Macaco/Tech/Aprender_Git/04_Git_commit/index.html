Actualizar conteúdos com o comando 'git' a partir do terminal

Uma forma de manter os dados a salvo.

Usamos várias opções do comando 'git'.
git add: que serve para adicionar os ficheiros à lista de actualizações.
git commit: que serve para confirmar as mudanças feitas
git push: que actualiza o nosso repositório no site de hospedagem.
