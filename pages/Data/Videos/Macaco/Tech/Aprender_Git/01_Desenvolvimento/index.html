Macaco Preguiçoso - GitHub e GitLab

Chamados Hubs tais como o github, gitlab, entre outros!

O Github é uma das plataformas de desenvolvimento de código mais popular.
É a partir desta plataforma que podemos desenvolver programas, como também ajudar ao desenvolvimento de outros programas que são maioritariamente de código Livre.
Existe o Gitlab o Codeberg, entre outras soluções igualmente gratuitas.
Pode-se usar estas plataformas para testar software para todas os sistemas operativos.
O mais comum é para Linux mas também existem projectos para Windiows que são de código Livre.
