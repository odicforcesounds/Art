Lista de videos sobre o uso do git.

Todos os vídeos estão em páginas únicas,
de forma a ser possível acrescentar mais informação a cada um deles, assim que possível.

Nota: Esta lista está incompleta, mas creio que partilho o necessário para que qualquer pessoa consiga aprender a partir da mesma.

01: Desenvolvimento: Apresentação de plataformas para desenvolvimento de software.

02: Github.com: Apresentação do site github.com e como fazer o registro.

03: Demonstração: Apresentação do uso da aplicação git a partir do site github.com.

04: git: Apresentação da rotina básica de actualização do projecto a partir do terminal.

05: Git Pages: Usar o github para fazer hospedagem do nosso site.

06: Git Clone: Copiar ou colonar um projecto que esteja no github.

07: Git Pull: Actualizar projecto local, sincronizando os dados com o projecto remoto.

08: Git Server: Usar a aplicação git como servidor pessoal, substituindo as plataformas online.
