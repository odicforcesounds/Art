O 'github' e o 'git'

Nada como um exemplo para exemplificar...

O git é uma aplicação que permite gerir projectos, a partir da sua função que atribui uma referência única a cada mudança.
Esta referência é alfa numérica e é ela que permite gerir o nosso projecto.
O github também permite fazer esta gestão a partir da web, mas o nosso objectivo é usar o Linux.
